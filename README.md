# goorm_study

- 자바 및 스프링 실력 향상을 위한 구름 백엔드 10회차 스터디입니다.
- 팀원들과 도서를 선정하여 학습 후 내용을 공유하는 것이 목표입니다.
- ⚠️ 부득이한 연유로 스터디에 참석하지 못하는 경우, 노션 캘린더에 미리 입력해주셔야 합니다.
